# **App Name**: SHIFT: Student Wellness Movement

## Core Features:

- Home Page: Landing page with a clear mission statement and the SHIFT values.
- Interactive Student Stories Map: Interactive map displaying student stories and experiences with swapping drinks, submitted via form.
- Healthier Alternatives 'Recipe Book': A student drink 'recipe book' featuring better drink alternatives. The drink mixes can be submitted by users for later approval.
- AI-Powered Drink Suggestions: AI-powered 'healthier swap' tool suggesting alternative drinks based on user input (e.g., favorite flavors, sugar content preferences). The tool can suggest recipe from the 'Recipe Book' or new drinks. The LLM will use a tool to decide when or if to suggest recipes from the 'Recipe Book'.
- Community Events Calendar: Monthly event calendar showcasing local swap events with juice vendors and health experts.
- Student Testimonials: Testimonial section displaying real-life stories of student swaps with the effects.

## Style Guidelines:

- Primary color: A muted forest green (#344E41), evoking nature and well-being.
- Background color: A light beige (#F6F1E9), providing a warm and inviting backdrop.
- Accent color: A soft gold (#D9C5AA), used sparingly for calls to action and highlighting important information.
- Body font: 'Open Sans', a humanist sans-serif that combines a modern look and warmth.
- Headline font: 'Playfair Display', a serif font with an elegant feel; use 'Open Sans' for body text if the headline font is paired.
- Simple, modern icons representing different drink types and health benefits, using a consistent line weight and the soft gold accent color.
- Clean, minimalist layout with clear sections, intuitive navigation, and ample white space to avoid overwhelming users.
